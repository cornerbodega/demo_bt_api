You can format your text using BBCode tags. Valid tags are:
  • [b] bold text [/b]
  • [i] italic text [/i]
  • [u] underlined text [/u]
  • [url] http://example.org[/url]
  • [url=http://example.org] Example [/url]
  • [img] http://example.org/image.jpg [/img]
  • [email] user@example.com [/email]
  • [quote] quoted text [/quote]
  • [quote=whom] quoted text [/quote]
  • [code] code block [/code]
Use [list=1] for numbered list items or [list=a] for alphabetic list items.
